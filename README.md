## 📌Interactive FAQ Page

This project is an interactive FAQ-style webpage built using HTML, CSS, and JavaScript.
Users can click on questions to toggle the visibility of their answers. The project demonstrates object-oriented programming (OOP) concepts and dynamic DOM manipulation in JavaScript.

----

## 🧱 Features

The website contains the following sections:
* Clean, responsive FAQ layout.
* Clickable questions that expand/collapse answers.
* Visual cues (+ for collapsed, - for expanded) to indicate state.
* Smooth CSS transitions for answer visibility.
* Fully implemented using JavaScript classes (QAItem).

----

## 🧱 Page Structure
1. Header
* Displays a clear "FAQ" heading for the page.
2. FAQ Section
* Generated dynamically with JavaScript.
* Each FAQ item includes a question and answer pair.
* Toggling answers handled by the QAItem class.

----

## 🎨 Styling & Design Approach
* Questions highlighted with bold text, hover effects, and plus/minus icons.
* Answers hidden by default and displayed dynamically on click.
* Smooth expand/collapse using CSS transitions.
* Responsive layout adapts to different screen sizes.
